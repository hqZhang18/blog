逐步分析
很像定位属性，我们需要考虑X值和Y值。X:0和Y:0表示从元素的左上角开始，并从左上角开始移动。X:100%指的是元素右边，Y:100%指的是元素底部。
对于上面创建的路径，实际是创建了如下的点：
x: 0, y:100%
x: 50%, y: 0
x: 100%, y: 100%
这个简单路径开始于左下角，水平移动50%，并到达顶部位置，然后又水平移动到100%的位置，垂直向下回到底部，到达第三个坐标点。三角形就出来了。
