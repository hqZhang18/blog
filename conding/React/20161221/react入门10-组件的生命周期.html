react入门10-组件的生命周期

2016-12-21
技术►React

生命周期介绍
1.组件的生命周期可分为三个状态：

Mounting: 组件挂载，已插入真实 DOM

Updataing: 组件更新，正在被重新渲染
Unmounting: 组件移出，已经移出真实 DOM

2.组件的生命周期可分为四个阶段:
创建、实例化、更新、销毁

3.网易新闻列表页面

1.Mounting/组件挂载相关：

(1) componentWillMount

组件将要挂载。在render之前执行，但仅执行一次，即使多次重复渲染该组件，或者改变了组件的state
(2) componentDidMount
组件已经挂载。在render之后执行，同一个组件重复渲染只执行一次


2.Updataing/组件更新相关

(1) componentWillReceiveProps(object nextProps)

已加载组件收到新的props之前调用，注意组件初始化渲染时则不会执行

(2) shouldComponentUpdate (object nextProps, object nextState)

组件判断是滞重新渲染时调用。该接口实际是在组件接收到了新的 props 或者新的state的时候 会立即调用，
然后通过返回

(3) componentWillUpdata(object nextProps, object nextState)
组件将要更新

(4) componentDidUpdata(object prevProps, object prevState)
组件已经更新

3.Unmounting/组件移除相关

(1) componentWillUnmout

在组件要被移除之前的时间点触发，可以利用该方法执行一些必要的清理组件将要移除

4.生命周期中与props和state相关

(1) getDefaultProps 设置props属性默认值

(2) getInitialState 升值state属性初始值
